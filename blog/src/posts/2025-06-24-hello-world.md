# 안녕하세요!

새 블로그를 시작합니다. 기존 포트폴리오 사이트를 개인 블로그로 리뉴얼했어요.

## 사용한 기술 스택

- **Vite** — 빠른 빌드 도구
- **React 18** — UI 라이브러리
- **TypeScript** — 타입 안정성
- **Tailwind CSS** — 유틸리티 기반 스타일링
- **React Router v6** — 클라이언트 사이드 라우팅

## 마크다운 지원

포스트는 `src/posts/` 디렉터리에 `.md` 파일로 작성합니다.

### 코드 예시

```tsx
export default function Hello() {
  return <h1>Hello, World!</h1>;
}
```

### 프론트매터 형식

```
---
title: 포스트 제목
date: 2024-01-01
tags: [tag1, tag2]
description: 포스트 설명
---
```

앞으로 공부한 내용, 개발 일지, 생각들을 여기에 기록할 예정입니다.
